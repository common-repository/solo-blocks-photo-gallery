== Description ==

Solo Blocks Photo Gallery for WordPress Gutenberg Editor. It's really easy to use and powerful free photo gallery which makes your online portfolio looking awesome. You can set up and manage it within Gutenberg WordPress editor in minutes. The exclusive drag & drop feature allows you to change the photos structure on the fly. We paid a lot of attention to the plugin optimization. You shouldn't worry about the gallery loading time. The gallery is fully responsive.

== Installation ==

After downloading the ZIP file,

1. Log in to WordPress dashboard panel.
2. Go to Plugins Add > New > Upload.
3. Click Choose file (Browse) and select the downloaded zip file of the gallery.
4. Click Install Now button.
5. Click Activate Plugin button for activating the gallery plugin.

If the installation does not succeed, please contact us.
